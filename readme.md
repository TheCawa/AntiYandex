**Инструкция:**
1. Скачайте последнюю версию с GitHub.
2. Откройте программу.
3. Нажмите кнопку «Завершить процессы»
4. Нажмите кнопку «Поиск Яндекса»
5. Нажмите на «Удалить Яндекс» Даже если появится ошибка - возможно, файлы уже удалены.
